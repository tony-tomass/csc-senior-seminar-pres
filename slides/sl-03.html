MALVERTISING

The main way someone can get malware is from downloading an infected file, and this can be done easily through ads


Common Types

Ad Fraud
— Most common type, also known as scam ads
— An ad that tricks someone into paying for a fake product
— Often "too good to be true"

Pop-up Ads
— A type of ad that opens a new browser tab or window to another website
— Usually works using JavaScript code, but can also be achieved using only HTML and CSS
— Ranges from intrusive and annoying to dangerous and malicious

Other Types and Methods
— Drive-by Download = Automatic file downloads when sent to a malicious website without permission
— Clickjacking = Hiding malicious links under legimate products, typically through CSS techniques
— Fake Virus Alert = Pretending to be real anti-virus alerts, commonly used in tech support scams
— Much more, and very common to see multiple methods used in one ad
